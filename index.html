jQuery-Geo-Typing

Easiest way to enable Georgian Keyboard Georgian Keyboard Driver Using Jquery

$ git clone git@github.com:gejadze/jQuery-Geo-Typing.git

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Usage

$(document).ready(function() {
  $(".georgian").geo();
});

Or you can turn it off like:

$(document).ready(function() {
  $(".georgian").geo(turn: "off");
});
